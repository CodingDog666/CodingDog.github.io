VBA 資料操作關鍵字：掌握 Excel 自動化的核心技巧

在 VBA 中，操作 Excel 工作表與儲存格是自動化流程的核心。透過幾個關鍵字，我們可以靈活操控資料，讓 Excel 變成強大的自動化工具。本篇文章將介紹幾個最常用的 VBA 資料操作關鍵字，幫助你提升 VBA 程式設計的效率。

1. Set

Set 關鍵字用來將物件（如工作表或範圍）指派給變數。這個過程稱為「物件指派」，它允許你將工作表或儲存格區域儲存在變數中，方便後續操作。

  • 範例：
Dim ws As Worksheet
Set ws = ThisWorkbook.Sheets("Sheet1")

這段程式碼將目前工作簿中的 “Sheet1” 工作表指派給變數 ws，你可以後續操作該工作表。

2. Range

Range 是最常用來操作儲存格或區域的物件。它允許你選取、讀取或寫入 Excel 表中的資料。無論你是想要更新單一儲存格或範圍，都可以透過 Range 來完成。

Range("A1").Value = "Hello"
Range("B1:C5").Interior.Color = RGB(255, 0, 0)

此範例分別將 “A1” 儲存格的值設為 “Hello”，並將 “B1” 區域的背景顏色設為紅色。

3. Cells

Cells 關鍵字則是用來操作個別儲存格。與 Range 不同，Cells 通常與迴圈結合來動態操作工作表中的多個儲存格。你可以透過 Cells(row, column) 來指定列與欄的位置。

  • 範例：
For i = 1 To 10
    Cells(i, 1).Value = i
Next i

這段程式會在 A1 到 A10 儲存格中依次填入 1 到 10。

4. Offset

Offset 關鍵字用來相對於某個參考位置進行移動。你可以從某個已選定的儲存格或範圍，根據偏移量來選取新範圍。

  • 範例：
Range("A1").Offset(1, 0).Value = "Next Row"

這段程式會在 “A1” 的下一行（即 A2）填入 “Next Row”。

Sub 自動填充資料()
    Dim ws As Worksheet
    Dim i As Integer

    ' 指派工作表
    Set ws = ThisWorkbook.Sheets("Sheet1")

    ' 循環填充 A1:A10
    For i = 1 To 10
        ws.Cells(i, 1).Value = i
        ws.Cells(i, 2).Value = i * i  ' 填入平方值
    Next i
End Sub

在這個範例中，Set 用來指定工作表，Cells 用來操作儲存格，並透過 For 迴圈自動填充 A1 到 A10 的資料，同時將平方值填入相對應的 B 列。

小結

掌握這些資料操作關鍵字能讓你更靈活地操控 Excel 工作表和資料。無論是 Set 指派物件、使用 Range 選取區域、透過 Cells 操作儲存格，還是利用 Offset 來進行相對位置的移動，這些關鍵字都是 VBA 自動化的核心工具。
